PTCGO-legacy环境介绍

发表于 2019-07-06 | 分类于 PTCG |

legacy是PTCGO一个较为特殊的环境，自其出现以来至今卡池从未变动，容纳HGSS-BW的所有卡牌。
本文介绍legacy中常用的一些万金油卡牌，以供交流。

泛用卡牌

Pokemon

Cleffa


无需能量进行一次滤抽，且无撤退费用，并配合poke-body具有一定防御能力，但有着醒来和到自己回合没醒无法行动的风险；
可适用于对场地有要求的慢速卡组（无需其他场地的慢速卡组可完全被tropical beach代替）。

Pichu


无需能量直接从卡组拉basic宝可梦到后场，但也会给对方铺场，与皮宝宝相同的无撤退费用、poke-body和招式附加效果是把双刃剑；
适合急需铺场的卡组。

Smeargle


poke-power几乎相当于多一次的supporter卡机会，极大地增大过牌量，虽然偶尔可能坑到自己，但综合来说利是远大于弊的；
适合有有效撤退措施的卡组。

Keldeo-EX


冲到前场的特性配合上float stone之类相当于每回合一次的无损换位；
适合经常需要换位和有撤退措施的卡组。

Exeggcute


特性可以从弃牌堆回收此卡，可以减少高级球、垃圾臂等的消耗损失；
适合需要经常弃牌的卡组。

Mr. Mime


特性能够保护后场的Pokemon不受伤害；
适合后场薄弱，需要保护的卡组。

Jirachi-EX


发动特性能够从直接卡组里检索一张supporter卡，一定程度上解决卡手问题，但上场后无用，并且身为HP最少的EX（也是唯一能被level ball检索的EX），容易被抓取两张prize card。
适合场不紧张，有多余空位的卡组。

Trainer

Item

Junk Arm


弃两张牌从弃牌堆拿回一张Item卡，非常强力的回收神器。

Pokemon Catcher/Pokémon Reversal


抓取对方后场的Pokemon，击杀对方的关键Pokemon可能对战局的影响很大。

Random Receiver


偶尔用来救命解决卡手的牌，还可以用junk arm回收利用。

Super Rod


回收弃牌堆中的Pokemon和basic energy到卡组，以达到续航的目的。

Switch


无视撤退费用与异常状态进行一次交换，且不计入撤退次数，可缓解危机。

Tool Scrapper


拆除场上最多两张tool card，限制对方。

Ultra Ball


最强的球，弃二可检索任何一只Pokemon。

Supporter

Colress


场上Pokemon越多就能抽滤越多的牌，中后期能形成大量过牌。

N


滤抽的同时影响对方，在prize card劣势时使用能限制对方。

Professor Juniper


稳定抽7，但要弃光手上的牌需要斟酌，同时一些情况下可堆墓。

Professor Oak’s New Theory


最稳定的滤牌，保留住好卡的同时过牌。

Skyla


几乎全卡检索牌，拿球≈Pokemon，拿energy相关≈energy，trainer可直接拿。

构筑思路

  • 主力输出Pokemon根据情况放3~4张
  • 辅助Pokemon若必要，但只需1只则放2张，防卡奖励或意外被击杀
  • 若某卡偶尔能起奇效，但不是一定能用到则放1张，可用super rod/junk arm回收
  • 场地根据情况放2~3张
  • 滤抽supporter（阿克罗玛、N、红豆杉博士、大木博士）一般放10张左右
  • 需要堆墓的卡组可多放红豆杉博士
  • 速度偏慢的卡组可多放N/阿克罗玛
  • 速攻快速击杀卡组建议少放N
  • 小人牌卡组一般带silver bangle/mirror
  • 需要快速铺场的卡组可带Pokemon collector
  • 容易处于prize劣势的卡组可带twins
